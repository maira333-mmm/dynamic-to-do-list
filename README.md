# 📖 About

This is an **Advanced To-Do List Application** designed to help users manage tasks efficiently. It goes beyond a basic to-do list by incorporating:

- **Task Prioritization** (Low, Medium, High)
- **Due Date Management**
- **Task Filtering** (All, Active, Completed)
- **Inline Task Editing**
- **Drag-and-Drop Reordering**
- **Local Storage** for data persistence

The application features a modern gradient design with smooth animations and a user-friendly interface.

---

# ✨ Features

| Feature | Description |
|----------|-------------|
| ✅ **Add Tasks** | Add new tasks with descriptions |
| ⏰ **Due Dates** | Set deadlines for each task |
| 🎯 **Priority Levels** | Low, Medium, High priority markers |
| ✏️ **Edit Tasks** | Double-click to edit task text |
| 🗑️ **Delete Tasks** | Remove tasks with one click |
| ✅ **Mark Complete** | Toggle task completion status |
| 🔍 **Filter Tasks** | View All, Active, or Completed tasks |
| 🔄 **Drag & Drop** | Reorder tasks by dragging |
| 💾 **Auto-Save** | Data persists in browser localStorage |
| 📱 **Responsive** | Works on all devices |

# 🚀 Getting Started

## 📋 Requirements

- 🌐 Modern Web Browser
- 💻 Visual Studio Code (Recommended)
- 🖼 Image files for icons (if customizing)

---

## 1️⃣ Clone Repository

```bash
git clone https://github.com/maira333-mmm/todo-list-app.git

cd todo-list-app
```

---

## 2️⃣ Open the Application

```text
Method 1:
Double-click index.html

Method 2:
Open the project using Live Server in Visual Studio Code

Method 3:
Open index.html in any modern web browser
```

---

## 3️⃣ Start Using the App

### ➕ Add a Task

- Enter the task description
- Select a due date (optional)
- Choose a priority level
- Click **Add**

### ✅ Manage Tasks

- Click a task to mark it as completed
- Double-click a task to edit it
- Click **✕** to delete it
- Drag & Drop tasks to reorder them

### 🔍 Filter Tasks

- **All** → Show every task
- **Active** → Show incomplete tasks
- **Completed** → Show completed tasks
